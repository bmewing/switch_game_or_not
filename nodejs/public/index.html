You might be wondering what the heck this is all about. I'm a big fan of the CAGCast Podcast hosted by CheapyD, Wombat and Shipwreck. Sometime in 2018 they started a segment called 'Switch Game or Not' where CheapyD would take the current week's new releases and insert fake game names. I thought it would be fun to build a simple AI bot to generate fake Switch names and formalize it into a website. If you want, you can see the code for the project on my GitHub repo and feel free to follow me on Twitter @pofigster.
